# 个人网页 - 单页面应用

一个现代化的个人展示网页，采用深色主题设计，包含个人介绍、作品展示、经历时间线、技能展示和AI对话功能。

## 功能特点

- 🎨 **现代设计**：深色主题配合渐变色彩，视觉效果出色
- 📱 **响应式布局**：完美适配桌面端和移动端
- 🎭 **流畅动画**：丰富的交互动画和过渡效果
- 💬 **AI对话**：智能AI助手，可以回答关于你的问题
- 🖼️ **作品轮播**：3D效果的图片轮播展示
- ⏱️ **时间线**：优雅的经历时间线展示
- 📊 **技能展示**：环形进度条展示技能熟练度

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可
2. 所有资源都通过CDN加载，无需安装任何依赖

## 自定义内容

### 修改个人信息

编辑 `index.html` 文件中的以下部分：

- **Hero区域**：修改 `.hero-title` 和 `.hero-subtitle` 中的文字
- **关于我**：修改 `.about-text` 中的介绍文字
- **照片**：替换 `about-image` 的 `src` 属性为你的照片URL
- **作品展示**：修改 `.carousel-item` 中的图片和标题
- **经历时间线**：修改 `.timeline-item` 中的内容
- **技能**：修改 `.skill-card` 中的技能名称和进度值

### 修改社交媒体链接

在 `index.html` 的 `.social-link` 元素中，将 `href="#"` 替换为你的实际链接。

### 自定义AI回复

编辑 `script.js` 文件中的 `generateResponse()` 方法，添加或修改AI的回复逻辑。

## 技术栈
